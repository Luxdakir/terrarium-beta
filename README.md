# terrarium-beta
> A NeoForge 1.21.1 mod that generates the entire Earth as a playable Minecraft world. Terrain, biomes, and climate are derived from real-world geographic datasets. Features include a longitude-based day/night cycle, polar nights and midnight sun, altitude effects, real-time weather integration, and support for Biomes O' Plenty.
>
> This project is a fork and extended port of the original Terrarium mod by Gegy, (https://github.com/Gegy/Terrarium) ported to NeoForge 1.21.1 with new geographic and atmospheric features.
